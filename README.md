## Mirror - API Chat
Mirror is a **free** **open-source** chat-only application compatible with [openai](https://openai.com/api/) and [openrouter](https://openrouter.ai/) api keys, with custom in-built tools for memory and project management.

### ⦿ Limitations:
- no image prompts - it cannot make images or see your images, there is no upload.
- no voice prompts - there is no audio/mic models, so you cannot talk to it.
- no chatgpt import - after careful consideration, the app is more stable without importing data from chatgpt. You may still copy/paste instructions and data.

*You are welcome to fork this repo to add these features!*

### ⦿ No server, no accounts, no signup.
It uses fully offline local storage for your chats, so you are responsible for your own backups via the "Backup All Data" button. Chats will be remembered as long as you are using the same device, and as long as you don't clear the app data yourself. It is recommended to save the backup in your google drive once you have it setup the way you want - the backup file matters more than the chat history.

### ⦿ What's the point if it's offline storage?
This is more of a chat app than a workspace. Mirror was originally created for creative writing but it is also suitable for those who enjoy a personal friend like 4o before it was deprecated. I wanted an app that handled memory, context, and lookup tools - without paying for a subscription hosting service.
